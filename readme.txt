The Showcase Social Media (icons) plugin enables you to easily display social media icons anywhere on your WordPress website via a shortcode.

== Description ==

The Showcase Social Media (icons) plugin is a simple, lightweight plugin designed to allow you to display visually appealing social media icons on your website. Whether you run an e-commerce platform, a service-providing website, or an informative blog, this plugin offers a streamlined way to show the social media you are on with a link.

Features:
- Supports multiple social media icons such as Facebook, Twitter, Instagram and many more.
- Simple settings page within the WordPress admin to choose which icons you want to display.
- Shortcode support for flexible placement on pages, posts, or widgets.
- Customizable icon sizes and spacing (in pixels) for seamless integration with your site design.
- Multiple icon styles available.
- Decide the order of your icons.

== Installation ==

1. Upload the showcase-social-media-plugin folder to the /wp-content/plugins/ directory, or install the plugin directly via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Fill in the identifiers you want to be displayed and in which order
3. Use the Social Icons settings page to further configure which icons are displayed, how they are displayed and where they need to link to.
4. Add the shortcode [ssmi_icons] to any page, post, or widget where you want the icons to appear.

== Frequently Asked Questions ==

= Can I adjust the size of the icons? =

Yes, you can adjust the size of the social media icons in pixels. The default is set to 55px.

= Can I adjust the spacing between the icons? =

Yes, you can adjust the spacing between the social media icons in pixels. The default is set to 1px. You can also use negative spacing which can come in handy with the first two styles of icons.

= What's the shape of the icons? =

You can choose between 4 styles. No shape in color or black, circular shape in either color or black.

= What social media icons are included? =

We currently have icons for the following: Behance, Discord, Dribble, Dropbox, Facebook, Flickr, Google Drive, Instagram, Kik, LinkedIn, LINE, Messenger, Patreon, PayPal, Pinterest, Reddit, Skype, Snapchat, Spotify, Threads, TikTok, Twitch, Twitter, Vimeo, WhatsApp, WordPress, YouTube.

= Does this plugin work with every WordPress theme? =

The plugin is designed to work with most themes, but we cannot guarantee perfect compatibility due to the diversity of themes available.
